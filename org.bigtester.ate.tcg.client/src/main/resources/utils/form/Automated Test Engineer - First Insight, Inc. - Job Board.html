Automated Test Engineer

Location: Warrendale, PA

Type: Full Time

Min. Experience: Experienced

First Insight, Inc. is seeking a well-qualified and experienced Test Engineer with some experience and interest in automated testing. This candidate’s primary responsibility will be to execute high-priority manual tests and help develop and execute automated tests for First Insight’s consumer engagement tools as part of a cross-functional agile team.

Help select and develop appropriate test automation tools, applying the latest techniques in test automation.
Perform manual static testing as needed.
Participate in prioritization of bugs in release cycles.
Create and maintain automated nightly build tests.
Participate in the design and test development of new and existing software solutions.
Work with offshore testing team and provide support as needed.
Participate and become a key player on R&D projects as needed.

 

Formal Education & Certification

University degree in the field of computer science, information systems, or another rigorous discipline is required. An advanced degree in one these fields is a plus.

 

Knowledge & Experience

  • 3+ years work experience with Java.
  • 3+ years work experience with HTML, CSS, and JavaScript.
  • Relevant work experience in development and/or testing role.
  • Familiarity with automated testing tools such as Selenium; proficiency with these skills a plus.
  • Familiarity with unit testing tools such as TestNG or JUnit; proficiency with these tools a plus
  • Basic database skills.
  • Good experience with different operating systems, with emphasis on Unix variants.
  • Good understanding of object oriented programming.
  • Ability to debug and resolve difficult technical problems.
  • Ability to determine the feasibility of projects in a fast paced deadline driven environment.
  • Ability to anticipate problems proactively and take appropriate actions to mitigate impact.
  • Experience with 3rd party custom software integration is a plus.
  • Exposure to iterative and incremental agile software development methodology.
  •  

Personal Attributes

  • Ability to set and manage priorities judiciously.
  • Ability to work under pressure.
  • Focus on the important testing and project priorities
  • Effectively balance curiosity about tests, test results, and bugs with project constraints on budget and schedule.
  • Understand the role of testing within the software development cycle while effectively advocating for the best possible customer experience.
  • Excellent written and oral communication skills.
  • Exceptionally self-motivated and directed.
  • Keen attention to detail.
  • Superior analytical, evaluative, and problem-solving abilities.
